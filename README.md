## Installation

```bash
$ npm install
```

## Running the app

```bash
1. git clone <this repository link>
2. npm i
3. put your variables in .development.env file

4. npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
